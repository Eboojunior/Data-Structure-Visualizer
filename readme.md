# DataStructureVisualizer

## Description

DataStructureVisualizer est une application en C++ utilisant **ImGui** pour visualiser différentes structures de données (telles que les tableaux, piles, listes, arbres binaires, etc.) de manière interactive avec une interface graphique.

## Fonctionnalités

- Visualisation des **tableaux** et **listes**.
- Visualisation des **piles** et **files d'attente**.
- Visualisation des **arbres binaires** et autres structures d'arbres.
- Interface utilisateur simple et réactive.
